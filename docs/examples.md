# Terminal UI Examples

This document showcases the terminal animations available in the core toolset. 
*Note: All animated GIF assets displayed below are stored locally in the `docs/assets/` directory.*

---

### 1. Interactive Selection Menu
![Interactive List](./assets/simple_list.gif)

Generates an inline menu where users can navigate and select from a predefined list of options using their keyboard arrow keys.

---

### 2. Determinate Progress Bar
![Progress Bar](./assets/progress.gif)

A dynamic visual indicator that tracks and updates the completion percentage of an ongoing synchronous task.

---

### 3. Indeterminate Loading Spinner
![Loading Spinner](./assets/spinner.gif)

A minimalist text animation indicating that a background process (like a network request) is active and running.
